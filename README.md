# Console-Based To-Do List

A command-line To-Do List app written in Python for CSFCP Assignment.

## Features
- Add, view, and remove tasks
- Simple menu navigation

## How to Run
1. Clone this repo
2. Run: `python todo.py`
3. Use menu prompts
